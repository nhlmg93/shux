## From Source

shux is built with Go. You can build it from source:

```bash
git clone https://github.com/nhelmig/shux.git
cd shux
go build ./...
```

The binary will be placed at `./shux`.

## Running

```bash
./shux
```

This starts shux in the foreground with default settings.
