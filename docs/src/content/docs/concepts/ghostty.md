## Real Terminal Emulation

Unlike traditional multiplexers (tmux, screen) that emulate a terminal in software, shux uses [Ghostty](https://ghostty.org) вҖ” a modern, GPU-accelerated terminal emulator вҖ” to provide **real terminal behavior** in every pane.

## Why Ghostty?

| Feature | tmux (emulated) | shux + Ghostty (real) |
|---------|----------------|----------------------|
| Colors | Limited palette | Full 24-bit color |
| Fonts | Monospace only | System fonts, ligatures |
| Unicode | Partial | Full Unicode support |
| Graphics | None | Image protocol, OSC 8 links |
| GPU | CPU-rendered | Hardware accelerated |
| Paste | Buffer-based | Native paste handling |

## Benefits

1. **Accurate emulation** вҖ” what you see is exactly what a real terminal would show
2. **Modern features** вҖ” ligatures, true type fonts, image rendering
3. **Performance** вҖ” GPU-accelerated rendering
4. **Compatibility** вҖ” works correctly with all terminal applications
